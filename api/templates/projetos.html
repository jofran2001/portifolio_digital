Aqui você encontra alguns dos projetos que desenvolvi

Projeto API

Durante o curso de graduação, desenvolvi um projeto cujo objetivo era integrar dados de uma estufa de alface com um site web, permitindo a visualização das informações da estufa através de gráficos dinâmicos e interativos. Este projeto envolveu a criação de uma API (Interface de Programação de Aplicações) que coleta e processa os dados da estufa, além de um site web que exibe essas informações de maneira intuitiva e acessível.

Projeto Dev Web

Durante o primeiro semestre da minha graduação, desenvolvi um site sobre a saga de games Fallout. O site contém a implementação do Flask, do banco de dados já integrado e funcionando, containers do docker e Bootstrap.

Criação do Design da Bolacha do Esquadrão de Segurança das Instalações

Desenvolvi o design da bolacha operacional do Esquadrão de Segurança das Instalações, selecionando cores ideais e dentro da norma estabelecida pelo Comando da Aeronáutica

Criação do Sistema de Identificação de Militares do GSD-SJ

Colaborei com o desenvolvimento do sistema que é usado para obter as informações de todos os militares do GSD-SJ (Grupo de segurança e Defesa de São José dos Campos)
